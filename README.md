# Budget V.

A **.NET** API to track the budget.

## Requirements
* .NET Core SDK 3.0

## Installation

Clone the project into your working directory:

``` bash
$ git clone https://github.com/kmhoran/BudgetV2.git .
```

## Usage

From the working directory, build and run:

``` bash
$ dotnet build
$ cd WebApi
$ dotnet run
```
